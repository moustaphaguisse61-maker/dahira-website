# DAHIRA MAFATIFOUL BICHRI - Site d'Inscription

## Description
Site web responsive pour l'inscription à la dahira MAFATIFOUL BICHRI. Champs: Nom, Prénom, Téléphone (10 chiffres). Validation client-side, stockage local (démo), message de succès.

Design moderne avec animations, gradient, glassmorphism. Prêt pour hébergement statique en ligne.

## Utilisation Locale
1. Ouvrir `index.html` dans un navigateur.
   - Windows: Double-clic ou `start index.html`
2. Remplir le formulaire et soumettre.
3. Vérifier console (F12) et localStorage pour les données.

## Déploiement en Ligne (Gratuit)
### Netlify (Recommandé - 1 clic)
1. Allez sur [netlify.com](https://netlify.com)
2. Glissez-déposez le dossier `dahira-website`
3. Site live instantané ! Ex: `https://votre-site.netlify.app`

### GitHub Pages
1. Créez repo GitHub, push dossier.
2. Settings > Pages > Deploy from branch `main` > `/root`
3. Live: `username.github.io/repo`
